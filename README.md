# أثَر | ملحمة كربلاء 🏴

تطبيق ويب تفاعلي شامل يهدف إلى توثيق وسرد واقعة الطف الخالدة (ملحمة كربلاء) استناداً إلى أمهات الكتب والمصادر التاريخية المعتبرة (مثل كتاب بحار الأنوار). يقدم التطبيق تجربة مستخدم روحانية وتاريخية فريدة تجمع بين أصالة المحتوى وحداثة التصميم التفاعلي.

## 🌟 الميزات الرئيسية

### 1. 📜 سجل الشهداء (أصحاب الحسين)
قاعدة بيانات تفاعلية متكاملة تضم تراجم مفصلة لشهداء كربلاء، مع إمكانية البحث والتصفية.
* **تصنيفات دقيقة:** أهل البيت (عليهم السلام)، الأصحاب الأنصار، شهداء ما قبل الطف (مثل سفراء الإمام)، النساء والشهداء الآخرون.
* **تفاصيل السيرة:** استعراض اسم الشهيد، لقبه، كيفية استشهاده، موعد الشهادة، قاتله، ونبذة تفصيلية عن حياته ومواقفه.
* **شجرة الأنساب:** عرض بصري لصلات القرابة والنسب بين شهداء بني هاشم.

### 2. 🖋️ ديوان الطف (النصوص والمواقف)
مكتبة غنية توثق الجانب الأدبي والخطابي والقصصي لنهضة الإمام الحسين (ع).
* **الخطب والأدعية:** خطب الإمام الحسين (ع) في مسيره، محاججات الأصحاب، وأدعية المعصومين.
* **الأراجيز والأشعار:** الأبيات التي ارتجزها الشهداء في ساحة المعركة، والمراثي الخالدة.
* **قصص ومواقف:** توثيق للمواقف البطولية، الكرامات، والأحداث التاريخية الدقيقة قبل وأثناء وبعد المعركة.
* **عرض النص الكامل:** واجهة قراءة مريحة تدعم النصوص الطويلة دون بتر أو اختصار، حفاظاً على الأمانة التاريخية.

### 3. 🕌 رواق الزيارات
قسم مخصص للزيارات المأثورة بأجواء روحانية، مع تصميم يحاكي القراءة من الصحف المكتوبة.
* نصوص الزيارات الموثقة بخطوط عربية أصيلة ومريحة للعين.
* تصميم يراعي الهدوء الروحاني والتركيز أثناء الزيارة.

### 4. 🗺️ خريطة المسير (مسيرة القافلة)
تتبع تفاعلي لرحلة ركب الإمام الحسين (ع) من المدينة المنورة إلى مكة المكرمة، وصولاً إلى كربلاء.
* استعراض المحطات التاريخية والمنازل التي توقف فيها الركب.
* ربط المواقف والأحداث بكل محطة جغرافية (مثل خطبته في ذي حسم، أو لقائه بالحر بن يزيد الرياحي).

### 5. 📊 الإحصائيات والتحليلات
رسوم بيانية توضح أرقاماً وإحصائيات متعلقة بواقعة الطف لتسهيل الاستيعاب التاريخي.
* توزيع الشهداء حسب الفئات والقبائل.
* التسلسل الزمني لشهداء ما قبل الطف (كالكوفة) ويوم عاشوراء.

### 6. ⚙️ إعدادات العرض المتقدمة (تجربة المستخدم)
* **الوضع الليلي والنهاري:** ألوان تم اختيارها بعناية لتعكس الأجواء الروحانية وتريح العين أثناء القراءة الطويلة.
* **التحكم بالخطوط:** إمكانية تكبير وتصغير حجم الخط، وتغيير نوع الخط ليناسب تفضيلات كل مستخدم.

## 🛠️ التقنيات المستخدمة

تم بناء هذا التطبيق باستخدام أحدث التقنيات لضمان أداء عالٍ وتجربة مستخدم سلسة:
* **React & Vite:** لبناء واجهات المستخدم وسرعة التطوير.
* **TypeScript:** لضمان جودة الكود وخلوه من الأخطاء.
* **Tailwind CSS:** لتصميم واجهات عصرية ومتجاوبة بالكامل.
* **Framer Motion:** لإضافة حركات انتقالية (Animations) ناعمة تعزز التفاعل.
* **Lucide React:** لتوفير أيقونات معبرة وأنيقة.

## 🚀 طريقة التشغيل للمطورين

1. تثبيت الحزم البرمجية:
   ```bash
   npm install
   ```
2. تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```
3. بناء التطبيق للإنتاج:
   ```bash
   npm run build
   ```
